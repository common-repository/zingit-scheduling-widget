== Description ==

Scheduling widget is an Official Plugin maintained by Zingit Solutions team that helps the patients to request their appointments with the Chiropractic offices. Scheduling widget plugin provides the ease of access in installing the widget on the officeâ€™s website.

* At a glance, this plugin adds the following:
* The widget installation steps are reduced with the help of this Plugin.
* Once the Widget is installed admin has to enter the Unique ID sent via email by Zingit Team.
* Admin has the ability to choose from either of popover or banner widget.
* This plugin is compatible to all the WordPress versions.
* Widget is auto responsive.
